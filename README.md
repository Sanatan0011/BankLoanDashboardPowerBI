# Bank Loan Power BI Dashboard

## Project Overview
This project contains a comprehensive Power BI dashboard that analyzes financial loan data. It is designed to provide clear, actionable insights into loan portfolios, focusing on key performance indicators (KPIs) such as total loan applications, funded amounts, and amounts received.

## Files in the Repository
- **`bank_loan_data_insights.pbix`**: The main Power BI file containing the dashboard, data model, and all visualizations.
- **`financial_loan.csv`**: The raw dataset containing detailed records of the financial loans used to power the dashboard.

## Dashboard Highlights
The dashboard helps understand the health of the loan portfolio by analyzing:
- **Good vs. Bad Loans**: Tracking the percentage of good loans (fully paid or current) versus bad loans (charged off).
- **Loan Status**: Breakdown of loans by their current status.
- **Geographic and Demographic Analysis**: Insights into loan distribution across different regions, employment lengths, and loan purposes.
- **Trends Over Time**: Monthly tracking of total funded amounts and received payments.

## How to Use
1. Clone the repository to your local machine.
2. Ensure you have [Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed.
3. Open `bank_loan_data_insights.pbix` to interact with the visual reports.
4. If Power BI prompts you to refresh the data, ensure the `financial_loan.csv` file path in the Power Query Editor points to the correct location on your local machine.

## Technologies Used
- **Power BI**: For data visualization and modeling.
- **Data Source**: CSV format.
